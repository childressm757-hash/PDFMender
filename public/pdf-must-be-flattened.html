“PDF Must Be Flattened” — What It Means (and How to Fix It)

In court filing, “flattened” usually means the PDF must be static — not interactive. Interactive fields (AcroForm) or XFA can cause rejection in CM/ECF and state eFile portals.

Run Court Preflight Fix Interactive/XFA Now

Why courts require flattening

  • Interactive objects can fail automated intake systems
  • Static PDFs are easier to scan and archive reliably
  • Some portals block active content for security reasons
